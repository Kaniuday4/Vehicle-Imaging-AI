# ssd-plate_detection
This is a plate detection project with SSD(Single Shot Detector)

## Requirements
- Python3
- Opencv(version 4.0 or higher)
- Tensorflow(optional-required for tf_detect.py)

## Test
- run `python3 detect.py --image images/img1.jpg`
